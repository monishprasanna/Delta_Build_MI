# AI-Powered Insurance Claims Agent

This repository contains an Anti-Gravity AI-powered insurance claims approval application built with React and Vite. It features a modern, 2D high-contrast black-and-white flat design aesthetic, simulating the end-to-end process of autonomous claim adjustment, including identity verification, liveness image acquisition, ML analysis, and smart-contract payment logic.

## Application Features Overview
- **Step 1:** Multi-Factor Authentication (MFA) & NLP pre-screening
- **Step 2:** AI Image Acquisition with AR overlay guidance & dynamic target requests based on claim type (e.g. Major Accident vs. Stolen)
- **Step 3:** Mocked Sequential Machine Learning analysis simulating identity, damage, and fraud profiling
- **Step 4:** Fiat & Cryptocurrency payout calculation with mocked IPFS immutable log anchoring
- **Step 5:** Built-in Appeals process for human-in-the-loop review

## Instructions to Run Locally

To run this application on your local machine, follow these steps:

### Prerequisites
Make sure you have Node.js installed on your machine. You can download it from [nodejs.org](https://nodejs.org/).

### 1. Clone or Download the Repository
If you haven't already, clone this repository locally:
```bash
git clone https://github.com/monishprasanna/Delta_Build_MI.git
```

### 2. Navigate to the App Directory
The React application is located in the `claims-agent` folder:
```bash
cd claims-agent
```

### 3. Install Dependencies
Install all the necessary packages for the application:
```bash
npm install
```

### 4. Start the Development Server
Run the local Vite server:
```bash
npm run dev
```

### 5. Open in Browser
Once the server starts, it will provide a localhost URL. Open your web browser and navigate to:
```
http://localhost:5173
```

You can now use and test the application locally!
